# Workshop Booking Redesign

I rebuilt FOSSEE's workshop booking site in React. Mobile-first, modern, responsive.

## Setup

```bash
npm install
npm run dev
```

Opens at `http://localhost:3000`. Production build: `npm run build`

## What's in Here
- 4 React components (Home, Workshops, BookingForm, Navigation)
- Mobile-first responsive design
- Works great on phones
- Modern purple gradient design
- Accessible forms with proper labels
- Fast load times
